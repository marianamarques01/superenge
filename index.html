GIF dentro da mensagem (Outlook): use Inserir → Imagens → Este dispositivo (ou arraste o ficheiro para dentro do texto, sobre a mensagem). O botão Anexar envia o GIF como anexo em separado, não aparece inline no corpo. No Outlook clássico (aplicação Windows), animações em GIF por vezes ficam só no primeiro frame; na Web costuma animar melhor. Use Baixar GIF (só logo / coluna) para um ficheiro pequeno a hospedar e referenciar por URL na assinatura (inclui selos se estiverem visíveis no preview). Qualidade: o GIF é limitado a 256 cores e perde o texto vetorial do HTML — a exportação usa supersampling e fundo branco para aproximar o resultado; para nitidez total prefira Copiar HTML.
